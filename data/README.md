# Data Directory

This directory contains:

- `hipaa_regulations.pdf` - Original HIPAA PDF (download from HHS.gov)
- `*_linearized.txt` - Extracted text from PDF (generated)
- `*_chunks.json` - Parsed chunks (generated)

## Getting the HIPAA PDF

Download the Combined Regulation Text from:
https://www.hhs.gov/hipaa/for-professionals/privacy/laws-regulations/combined-regulation-text/index.html
